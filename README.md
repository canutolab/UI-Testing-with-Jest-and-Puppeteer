# Testing

## Getting Started

### Dependencies:
Tools needed to run this app: `node` and `npm`

### Installing:
* `npm install` to install dependencies

### Running the App:
After you have installed all dependencies, you may run the app.

- `node index`
- `npm run watch`
- `npm run watch -- --mode development`
- `npm run jest`
